"Meaningful Media"

Text, image, audio, video + interface as index
(meaning through indication)

Text, image, audio, video + interface as icon
(meaning through likeness or analogy)

Text, image, audio, video + interface as symbol

Rhetoric + Types of Media

"Persuasive Media"

Text, image, audio, video + interface as ethos
(persuasion through credibility or authority)

Text, image, audio, video + interface as pathos
(persuasion through feeling or emotion)

Text, image, audio, video + interface as logos
(persuasion through logic or reason)

Text, image, audio, video + interface as kairos
(persuasion through time or timing)
